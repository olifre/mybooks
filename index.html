My Books

This page contains some writing projects by me. The English stories have been written using the pseudonym Wilkie Goldentongue, the stories without pseudonym are written in German. All of these are original works of different genres and different quality.
You can go back to my main page by following this link to olifre.github.io.
If you are interested in the code producing all these formats, feel free to check out the repository on GitHub.

My Life in a Nutshell

— by Wilkie Goldentongue

The largest project so far. It is a blog-like mix of a diary and a novel, written in real-time during over a few months, at a time when the term "blog" did not really exist yet.
The language and layout have not been polished, and pseudonyms are used. It has a small greeting in each "post", which starts the chapter, and at the end of each chapter, some poetry.
Note that this is a very raw and unpolished work, which however might make it an interesting experience for a coming-of-age story.
It may still see updates from time to time.
